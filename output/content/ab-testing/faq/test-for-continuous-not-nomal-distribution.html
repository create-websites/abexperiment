Title: "Unlocking AB Testing: Non-Parametric Tests for Continuous Data"

Summary: The article discusses the use of non-parametric tests in AB testing with continuous data that is not normally distributed, highlighting tests like Mann-Whitney U, Wilcoxon Signed-Rank, Kruskal-Wallis, and Friedman for comparing groups' distributions and medians under such conditions.
TEST FOR CONTINUOUS NOT NOMAL
        



When conducting AB testing with continuous data that is not normally distributed, non-parametric tests are typically used. Non-parametric tests do not make assumptions about the underlying distribution of the data, making them suitable for non-normally distributed data. Some common non-parametric tests for AB testing with continuous data include:

Non-Parametric Test Description
Mann-Whitney U Test A non-parametric test used to compare the distributions of two independent groups. It is suitable for comparing the means of two groups when the data is not normally distributed.
Wilcoxon Signed-Rank Test Similar to the Mann-Whitney U Test, but used for paired data. It compares the medians of two related groups when the data is not normally distributed.
Kruskal-Wallis Test A non-parametric alternative to the one-way ANOVA test. It is used to compare the medians of three or more independent groups when the data is not normally distributed.
Friedman Test Similar to the Kruskal-Wallis Test, but used for related samples. It compares the medians of three or more related groups when the data is not normally distributed.
